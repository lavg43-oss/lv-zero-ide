# Términos de Uso — lv-zero

**Última actualización:** Junio 2026

## 1. Aceptación de Términos

Al usar lv-zero ("el Software"), aceptas estos términos en su totalidad. Si no estás de acuerdo, no uses el Software.

## 2. Propósito del Software

lv-zero es una **herramienta de asistencia para desarrollo de software** de código abierto. Está diseñada para ayudar a usuarios a crear, modificar y entender código mediante inteligencia artificial.

El Software **NO es**:
- Un sistema autónomo de toma de decisiones
- Un reemplazo para juicio humano profesional
- Una herramienta certificada para industrias reguladas (salud, finanzas, aviación, etc.)

### 3.1 Responsabilidad del Usuario

El usuario es el **único responsable** por:

- **El código generado**: Revisa y verifica todo el código antes de usarlo en producción
- **El cumplimiento legal**: Asegúrate de que el uso del Software cumple con las leyes aplicables
- **La seguridad**: Implementa medidas de seguridad apropiadas para tu caso de uso
- **Datos sensibles**: No compartas información confidencial, contraseñas o datos personales a través del Software si no tienes las medidas de protección adecuadas

### 3.2 Contenido Generado por IA

El Software utiliza modelos de lenguaje de gran escala (LLM) que pueden generar contenido **incorrecto, incompleto o inapropiado**. El usuario debe:

- Verificar todo el código generado antes de ejecutarlo
- No asumir que el código generado es seguro, eficiente o libre de errores
- Realizar pruebas y revisiones de seguridad apropiadas

## 4. Uso Aceptable

No uses el Software para:

- Crear malware, virus o software malicioso
- Violar derechos de propiedad intelectual
- Realizar actividades ilegales
- Generar contenido dañino, discriminatorio o ilegal
- Eludir medidas de seguridad o sistemas de autenticación

## 5. Privacidad y Datos

- El Software se ejecuta localmente en tu máquina
- Las API keys se almacenan localmente (encriptadas)
- Las solicitudes a APIs externas (DeepSeek, OpenAI, etc.) se rigen por los términos de esos proveedores
- No recopilamos datos de uso personal identificables

## 6. Servicios de Terceros

lv-zero se conecta a servicios de terceros (DeepSeek, Google Workspace, Cloudflare, GitHub, etc.). El uso de estos servicios se rige por sus propios términos y condiciones. No somos responsables por:

- La disponibilidad o funcionamiento de servicios de terceros
- Los costos asociados al uso de APIs de terceros
- El manejo de datos por parte de terceros

## 7. Modificaciones

Estos términos pueden actualizarse periódicamente. El uso continuado del Software después de las modificaciones constituye la aceptación de los nuevos términos.
